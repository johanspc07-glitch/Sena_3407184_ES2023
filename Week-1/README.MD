**Dominio**: Plataformas Freelancers

## 🎯 Objetivo

Construir un script JavaScript que se ejecute en Node.js y muestre en la consola una **ficha de presentación** de una entidad del dominio **Plataformas Freelancers**, aplicando lo aprendido esta semana: `console.log()`, tipos primitivos (string, number, boolean) y comentarios.

No se utilizan variables todavía. Solo `console.log()`, tipos de datos, expresiones aritméticas y comentarios.

---

## 📋 Descripción

Se construirá un pequeño script que funcione como un **carnet digital de consola** para una plataforma de freelancers.

El programa mostrará información organizada sobre **un proyecto freelance**, que es una de las entidades principales dentro de una plataforma de trabajo independiente.

Ejemplos de información que puede mostrar el script:

- Nombre del proyecto
- Código del proyecto
- Categoría del trabajo
- Presupuesto del proyecto
- Estados del proyecto

---

## ✅ Requisitos

### Requisitos de contenido

El script debe mostrar **como mínimo**:

1. Nombre del **proyecto freelance** (string)
2. Identificador o código único del proyecto (string o number)
3. Categoría o tipo de trabajo (string)
4. Al menos **un dato numérico** relacionado con el proyecto (number)
5. Al menos **dos valores boolean** que representen estados del proyecto
6. Una **expresión aritmética** calculada (ej: presupuesto con comisión)
7. Una sección de **resumen del dominio**

### Requisitos técnicos

- [ ] El archivo se llama `script.js` y corre con `node script.js`
- [ ] Usa `console.log()` para toda la salida
- [ ] Incluye como mínimo **3 comentarios** que expliquen secciones del código
- [ ] La salida está organizada con separadores visuales
- [ ] Los comentarios están escritos en **español**
- [ ] No hay errores al ejecutar con `node`
