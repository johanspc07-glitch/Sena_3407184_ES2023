# Proyecto Semana 03: Calculadora de Dominio

## 🎯 Objetivo

Construir una **calculadora multi-operación** aplicada al dominio **Plataformas Freelancers**, usando operadores aritméticos, de asignación compuesta, comparación estricta y operadores lógicos.

El objetivo es simular cálculos reales dentro de una plataforma freelance, como ingresos por proyectos, comisiones y validaciones de condiciones.

---

## 📋 Dominio del Proyecto

**Dominio**: Plataformas Freelancers

Este dominio representa plataformas digitales donde:

- Clientes publican proyectos
- Freelancers ofrecen sus servicios
- Se manejan presupuestos, comisiones y estados de proyectos

Ejemplos de cálculos dentro de este dominio:

- 💻 Calcular ingresos por proyectos realizados
- 💰 Calcular comisión de la plataforma
- 📊 Determinar proyectos disponibles o pendientes
- ✅ Validar si un freelancer puede tomar un proyecto

---

## ✅ Requisitos Funcionales

La calculadora incluye:

1. **Operaciones aritméticas** (`+`, `-`, `*`, `/`, `%`)
   - Cálculo de ingresos totales y comisiones

2. **Operadores de asignación compuesta** (`+=`, `-=`, `*=`, `/=`)
   - Acumulación de ganancias por múltiples proyectos

3. **Comparación estricta** (`===`, `!==`, `>=`, `<=`, `>`, `<`)
   - Validación de proyectos completados o pendientes

4. **Operadores lógicos** (`&&`, `||`, `!`)
   - Validación de condiciones para aceptar proyectos

5. **Numeric separators**
   - Uso de `_` para mejorar la lectura de valores numéricos

6. **`console.log()`**
   - Resultados claros y organizados en consola
