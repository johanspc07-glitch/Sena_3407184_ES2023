# Proyecto Semanal — Semana 02: Ficha de Datos del Dominio

> 🎯 **ÚNICO ENTREGABLE**: Este proyecto es el **único entregable obligatorio** para aprobar la semana.

---

## 🎯 Objetivo

Crear una ficha de datos en consola usando variables y tipos de datos en **JavaScript**, adaptando el proyecto al dominio **Plataformas Freelancers**.

El objetivo es practicar el uso de:

- Variables con `const`
- Tipos de datos (`string`, `number`, `boolean`, `null`)
- Verificación de tipos con `typeof`
- Conversiones explícitas de datos

---

## 📋 Dominio del Proyecto

**Dominio**: Plataformas Freelancers

Las plataformas freelancers son sistemas digitales donde **clientes publican proyectos** y **profesionales independientes (freelancers)** ofrecen sus servicios para realizarlos.

En este proyecto se utiliza como entidad principal un **Proyecto Freelance**, el cual contiene información básica como:

- Nombre del proyecto
- Categoría del trabajo
- Presupuesto
- Estado del proyecto
